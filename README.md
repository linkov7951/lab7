```markdown
# Лабораторная работа №7

## Условия задач

### Задача 1
Реализовать функцию swap для обмена значений между аргументами x и y, не используя при этом третьей переменной.

### Задача 2
Заданы следующие функции:
- f1(x) = sin(x)/x
- f2(y) = y² + 2y^(3/2)
- f3(z) = (z+2)(z+1)z

Вычислить значение функции R(x, y, z) = f1(x) + f2(y) * f3(f2(z)) для x = 0.6, y = 0.3, z = 0.1.

Реализовать обобщенную функцию печати значения функции от аргумента:
`void print(float x, float (*y) (float))`

### Задача 3
Реализуйте перегрузку функций max и min для массива из k вещественных чисел. С помощью программы найдите произведение минимумов двух массивов различной длины и среднее геометрическое их максимумов.

### Задача 4
Составить алгоритм и программу, реализующую разные механизмы передачи параметров, на примере нахождения значений функций:
1. Функция принимает параметры по значению и возвращает значение
2. Функция в качестве параметров принимает указатели, в том числе и параметр адреса результата, а сама функция типа void

Функция вычисляет сумму членов ряда x^(2n-1)/(2n-1). Вычисление суммы необходимо выполнять до тех пор, пока очередное слагаемое по модулю не станет меньше заданной точности e = 0.5*10⁻⁴, x = 0.20.

## Файлы

- `zadanie2.c` - программа для вычисления функции R(x,y,z)
- `zadanie3.cpp` - программа с перегрузкой функций max и min
- `zadanie4.c` - программа для вычисления суммы ряда
```

```
### Задача 2
**Входные данные:** x = 0.6, y = 0.3, z = 0.1

**Вычисление:**
- f1(0.6) = sin(0.6)/0.6 = 0.941071
- f2(0.3) = 0.3² + 2*0.3^(3/2) = 0.09 + 2*0.1643 = 0.4186
- f2(0.1) = 0.1² + 2*0.1^(3/2) = 0.01 + 2*0.0316 = 0.0732
- f3(0.0732) = (2.0732)*(1.0732)*0.0732 = 0.1629
- f2(0.3) * f3(f2(0.1)) = 0.4186 * 0.1629 = 0.0682

**Результат:** R = 0.941071 + 0.0682 = **1.009271**

### Задача 3
**Входные данные:**
- Массив 1: {3.5, 1.2, 7.8, 2.3, 9.1}
- Массив 2: {4.1, 6.7, 0.9, 5.4}

**Результаты:**
- Минимум массива 1: 1.2
- Минимум массива 2: 0.9
- Произведение минимумов: 1.2 * 0.9 = 1.08
- Максимум массива 1: 9.1
- Максимум массива 2: 6.7
- Среднее геометрическое максимумов: √(9.1 * 6.7) = √60.97 = 7.81

**Выходные данные:**
```
1.08
7.81
```

### Задача 4
**Входные данные:** x = 0.20, e = 0.00005

**Вычисление ряда:**
- n = 1: 0.20¹/1 = 0.2
- n = 2: 0.20³/3 = 0.008/3 = 0.0026667
- n = 3: 0.20⁵/5 = 0.00032/5 = 0.000064
- n = 4: 0.20⁷/7 = 0.0000128/7 = 0.00000183 (меньше e, останов)

**Сумма:** 0.2 + 0.0026667 + 0.000064 = **0.202731**

**Выходные данные:**
```
0.202731
0.202731
